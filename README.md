# 🎬 AniStream

**Votre bibliothèque d'animes en streaming, intelligente et automatique.**

AniStream est une plateforme de streaming anime nouvelle génération qui **révolutionne l'importation d'épisodes**. Fini le copier-coller manuel des URLs — il suffit de sélectionner un anime pour que la plateforme scrape automatiquement **anime-sama.to** et récupère l'intégralité des épisodes disponibles. Simple, rapide, et sans friction.

🔗 **Accédez à AniStream :** [anistreamvf.github.io](https://anistreamvf.github.io)

---

## ✨ Fonctionnalités clés

| Fonctionnalité | Description |
|---|---|
| 📦 **Importation automatique** | Scraping intégré d'**anime-sama.to** : un clic, tous les épisodes |
| 🧠 **Cache intelligent** | Réduction de la latence et économie de bande passante grâce à une mise en cache optimisée |
| ⚡ **Serveurs adaptatifs** | Sélection automatique du meilleur serveur disponible pour une lecture sans coupure |
| 👤 **Profils utilisateurs** | Personnalisation de l'expérience et accès à vos animes favoris |
| 🎥 **Lecteur fluide** | Interface épurée, contrôles intuitifs et qualité adaptative |
| 🎉 **Watch Party** | Regardez vos animes préférés en synchronisation parfaite avec vos amis |
| 📊 **Suivi de progression** | Sauvegarde automatique des épisodes visionnés, où que vous soyez |
| 🔍 **Catalogue & Recherche** | Navigation avancée avec filtres pour trouver l'anime idéal en un instant |
| 📱 **Cross-Plateformes** | Installation en tant qu'application native sur mobile et desktop  |
