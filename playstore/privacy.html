Politique de confidentialité

Dernière mise à jour : juillet 2026

1. Introduction

La présente politique de confidentialité décrit comment l'application AniStream ("nous", "notre", "l'application") collecte, utilise et partage les informations des utilisateurs.

2. Données collectées

AniStream peut collecter les informations suivantes :

  • Informations de compte : nom d'utilisateur et adresse e-mail (si vous créez un compte).
  • Identifiant appareil : Android ID, utilisé uniquement pour le fonctionnement technique de l'application.
  • Données de préférences : paramètres de l'application et préférences de lecture stockés localement.

3. Utilisation des données

Les données collectées sont utilisées pour :

  • Fournir et maintenir le service de l'application
  • Personnaliser l'expérience utilisateur
  • Améliorer les fonctionnalités et les performances

4. Partage des données

AniStream ne vend pas les données personnelles de ses utilisateurs. Les données ne sont partagées qu'avec les services techniques nécessaires au fonctionnement de l'application (hébergement cloud, bases de données).

5. Sécurité

Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, modification ou suppression.

6. Suppression des données

Vous pouvez supprimer votre compte et toutes les données associées depuis les paramètres de l'application. Les données locales peuvent être effacées via les paramètres Android.

7. Contact

Pour toute question concernant cette politique de confidentialité : contact@anistream.app
